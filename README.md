# 🛒 E-Commerce Website

## 📌 Project Description

This is a simple **E-Commerce Website** built using **HTML, CSS, and basic JavaScript**.
The project simulates an online shopping interface where users can browse products and click a **Buy** button.

When the user clicks **Buy**, an alert message appears confirming the item has been added to the cart.

---

## 🚀 Features

* 🛍️ Product display section
* 🖱️ "Buy Now" button interaction
* 🔔 Alert message: *"Product added to cart"*
* 🎨 Styled using CSS
* ⚡ Simple and beginner-friendly project

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (for alert functionality)
